### Bug Fix: Infinite HITL Re-gating Loop

- **Root cause**: After HITL rejection, the ProposingAgent re-fires (no Hypotheses in context), re-proposes the same fact, triggers HITL gate again → infinite loop
- **Fix 1**: Engine tracks `rejected_proposals: HashSet<String>` — re-proposals with the same ID are silently discarded
- **Fix 2**: Diagnostic fact emitted to `ContextKey::Diagnostic` on rejection for agent observability
- Both fixes align with "No Hidden Work" and "Append-Only Truth" tenets

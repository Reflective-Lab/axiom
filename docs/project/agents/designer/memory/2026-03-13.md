## Cross-Cutting Observation: `color: white` Pattern

Found `color: white` instead of `var(--paper)` across all three new pages (Pricing, ForOperations, ObservationUI). The values are identical today (#f5f4f0 is paper, white is white — they're actually different). This should be `var(--paper)` for consistency. If we ever shift to a true white paper bg, this won't matter. But if paper stays warm off-white, accent buttons with `color: white` will look slightly different from the rest of the page.

**Recommendation:** Jules should do a project-wide search-replace of `color: white` → `color: var(--paper)` on all accent/tertiary buttons.
